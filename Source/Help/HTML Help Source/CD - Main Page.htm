UDP Custom Device Main Page

Introduction

When a UDP Custom Device is first instantiated in a Veristand system definition, this is the root-most landing page presented to the user. Functionality and features concerning the entire custom device are provided here.

Configuration Summary

These are configuration parameters that control the import/export of the custom device, versioning information, and global timestamping details. Importantly, the import and export functionality is currently non-functional and should not be used. It will be patched in a later release.

Description
Version Displays the current version of the UDP Custom Device installed.
Timestamp Channel Displays the timestamp channel to use for all transmit packets in the Custom Device. This channel may be modified and selected by clicking on the Select... button.
Export Configuration Present on the top menu and as a right-click menu from the tree. Exports the current UDP custom device configuration directly to a binary or XML file.
Import Configuration Present on the top menu and as a right-click menu from the tree. Imports the current UDP custom device configuration directly from a previously-exported binary or XML file.
