Receive Packets

Introduction

This page enumerates all of the receive packets that have been configured in the current system definition. It further provides a means to add or remove receive packets to the custom device.

Configuration Summary

The primary intent of this page is to display the current receive packets that have been defined. The ability to add or remove packets is enabled by this page, but the functions are provided by the buttons at the top of the page or by right-clicking on the listbox. Packets are enumerated in the order that they are expected to be received in.

Description
Rx Packets Displays all the receive packets defined for transaction. By selecting a packet and right-clicking on this control, it is possible to remove packets from the definition.
