Supported Data Types

Introduction

The UDP Custom Device, as of this time, only supports a small subset of the possible data types it can encode into a UDP stream and decode from a UDP stream. This page will cover the currently supported data types and any particular behaviors that may occur during encoding and decoding Veristand double channels.

Supported Data Types

Conversion Behavior
Boolean Transmit Encoding: Converts from a Veristand double channel, to a 8-bit representation of a boolean value. Any value not equal to 0 will be converted to a boolean high; a byte with a value of 1. Any value equal to 0 will be converted to a boolean low; a byte with a value of 0.
Receive Decoding: Converts from a 8-bit boolean to a 64-bit, double-precision, floating point Veristand channel value. A boolean high (a byte with anything other than a 0 value in it) will be converted to a 1. A boolean low (a byte that is all zeroes) will be converted to a 0.
Single Transmit Encoding: Converts from a Veristand double channel, to a 32-bit, single-precision, floating point value. Retains high levels of floating point accuracy, but the conversion times from a double to a single may be long in high channel counts transmissions.
Receive Decoding: Converts from a 32-bit single to a 64-bit, double-precision, floating point Veristand channel value. The resultant double after the conversion will still only be as precise as the single it is converting from.
Double Transmit Encoding: Maintains the encoding for a 64-bit floating-point double. This is the native format of all Veristand channels.
Receive Decoding: Maintains the encoding for a 64-bit floating-point double. This is the native format of all Veristand channels.
I8 Transmit Encoding: Converts from a Veristand double channel, to an 8-bit signed integer representation of the value. Any decimals will be coerced to 0.
Receive Decoding: Converts from an 8-bit signed integer to a 64-bit, double-precision, floating point Veristand channel value.
I16 Transmit Encoding: Converts from a Veristand double channel, to an 16-bit signed integer representation of the value. Any decimals will be coerced to 0.
Receive Decoding: Converts from a 16-bit signed integer to a 64-bit, double-precision, floating point Veristand channel value.
I32 Transmit Encoding: Converts from a Veristand double channel, to an 32-bit signed integer representation of the value. Any decimals will be coerced to 0.
Receive Decoding: Converts from a 32-bit signed integer to a 64-bit, double-precision, floating point Veristand channel value.
I64 Transmit Encoding: Converts from a Veristand double channel, to an 64-bit signed integer representation of the value. Any decimals will be coerced to 0.
Receive Decoding: Converts from a 64-bit signed integer to a 64-bit, double-precision, floating point Veristand channel value.
U8 Transmit Encoding: Converts from a Veristand double channel, to an 8-bit unsigned integer representation of the value. Any decimals will be coerced to 0. Negative values are coerced to 0.
Receive Decoding: Converts from an 8-bit unsigned integer to a 64-bit, double-precision, floating point Veristand channel value.
U16 Transmit Encoding: Converts from a Veristand double channel, to an 16-bit unsigned integer representation of the value. Any decimals will be coerced to 0. Negative values are coerced to 0.
Receive Decoding: Converts from a 16-bit unsigned integer to a 64-bit, double-precision, floating point Veristand channel value.
U32 Transmit Encoding: Converts from a Veristand double channel, to an 32-bit unsigned integer representation of the value. Any decimals will be coerced to 0. Negative values are coerced to 0.
Receive Decoding: Converts from a 32-bit unsigned integer to a 64-bit, double-precision, floating point Veristand channel value.
U64 Transmit Encoding: Converts from a Veristand double channel, to an 64-bit unsigned integer representation of the value. Any decimals will be coerced to 0. Negative values are coerced to 0.
Receive Decoding: Converts from a 64-bit unsigned integer to a 64-bit, double-precision, floating point Veristand channel value.
